# NoteNotch

[![Website](https://img.shields.io/badge/Website-notenotch.app-blue)](https://alexlazarian.github.io/notenotch.app/)
[![macOS](https://img.shields.io/badge/Platform-macOS%2012%2B-lightgrey)](https://alexlazarian.github.io/notenotch.app/)

The official website and release distribution for [NoteNotch](https://alexlazarian.github.io/notenotch.app/) - a floating teleprompter app for macOS.

## About

This repository contains:
- **Marketing website** - Built with React, TypeScript, and Tailwind CSS
- **App releases** - Distributed via GitHub Releases
- **Sparkle appcast** - For automatic app updates

## Website

Visit the live site: **[alexlazarian.github.io/notenotch.app](https://alexlazarian.github.io/notenotch.app/)**

## Development

```bash
cd web
npm install
npm run dev
```

## Deployment

The website automatically deploys to GitHub Pages when changes are pushed to `main`.
